Cascading Style Sheets  层叠样式表

网页的表现层


* css的几种表现形式：内敛、嵌入、外部链接

内嵌
  <head>
    <style type="text/css">
        css 代码
    </style>
  </head>

嵌入
  
    <div style="font-size:18px;"></div>   


外部链接
  
    <head>
      <link href="../css/reset.css" rel="stylesheet"/>
    </head>
